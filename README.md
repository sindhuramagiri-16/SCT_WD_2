# Stopwatch Web Application

## Overview
A responsive and interactive stopwatch web application built using HTML, CSS, and JavaScript.

## Features
- Start Stopwatch
- Pause Stopwatch
- Reset Stopwatch
- Record Lap Times
- Display Lap History
- Responsive UI

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Run
1. Download the project.
2. Open folder in VS Code.
3. Open index.html in browser.
